Aplicación para realizar transacciones por medio de Sinpe móvil


Desarrollé una aplicación móvil multiplataforma utilizando React Native, enfocada en simplificar y optimizar transacciones financieras para los usuarios. Este framework me permitió crear una interfaz intuitiva y consistente tanto en Android como en iOS, asegurando una experiencia de usuario ágil y responsiva. React Native fue clave para integrar funcionalidades como el manejo de listas de transacciones, botones interactivos y navegación fluida entre pantallas.
La aplicación se conecta con AWS DynamoDB, una base de datos NoSQL en la nube, para almacenar información esencial como balances de usuarios, historial de transacciones y contactos frecuentes. La elección de DynamoDB permitió un almacenamiento seguro, con alta disponibilidad y capacidad de escalar según el crecimiento de los datos. Además, se implementó un diseño eficiente para consultas rápidas y actualizaciones en tiempo real.
Para manejar la lógica del servidor, integré funciones AWS Lambda, encargadas de validar transacciones, procesar envíos de dinero y actualizar los balances de los usuarios. Estas funciones se ejecutan en la nube, asegurando escalabilidad y un manejo eficiente de los recursos. Gracias a Lambda, los usuarios reciben confirmaciones rápidas y consistentes de sus operaciones, lo que garantiza una experiencia confiable.
Este proyecto combina lo mejor de las tecnologías móviles y los servicios en la nube, demostrando mi habilidad para desarrollar soluciones completas e integradas. React Native me permitió crear una interfaz atractiva y funcional, mientras que AWS DynamoDB y Lambda aseguraron un backend robusto y escalable, logrando una aplicación eficiente y lista para el futuro.
