Generador de Títulos en PDF


El proyecto consiste en el desarrollo de una aplicación de escritorio para Windows creada con Electron, orientada a la generación automatizada de títulos en formato PDF.
La aplicación permite ingresar una imagen base que funciona como plantilla visual para el diseño del documento.
Asimismo, se carga un archivo Excel que contiene la información necesaria, como nombre, apellidos, número de cédula y número de teléfono.

A partir de estos datos, el sistema genera automáticamente múltiples archivos PDF personalizados. Este proceso garantiza rapidez, precisión y uniformidad en la presentación de los documentos. Los archivos PDF generados deben guardarse manualmente en una carpeta definida por el usuario. Posteriormente, estos documentos se almacenan en una carpeta de Google Drive para su distribución. La aplicación compara los nombres de los archivos PDF con los registros del archivo Excel. Este mecanismo permite identificar de forma precisa a qué contacto corresponde cada documento. Con base en esta comparación, el sistema prepara el envío automático de los archivos.

El envío se realiza mediante la aplicación de escritorio de WhatsApp, optimizando la comunicación. Este flujo automatizado reduce significativamente el tiempo de trabajo y el esfuerzo manual. Además, se minimiza la probabilidad de errores en la asignación y envío de los documentos. La interfaz gráfica es intuitiva, clara y fácil de utilizar para cualquier tipo de usuario. En conclusión, la aplicación representa una solución moderna, eficiente y confiable para la generación y distribución de títulos personalizados.
